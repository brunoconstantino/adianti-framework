Adianti Framework is installed!
Now, you can start to create your application, following these steps:

1) Create a database for your project;
2) Create model classes to represent your entities;
3) Create the application controllers;
4) Customize the templates.

Use the Adianti Studio Professional to speed up the creation of your application. Studio Pro allows you to design your interfaces (forms) for Adianti Framework, as well as PDF documents that can be used for report generation. Studio Pro also allows you to automatically generate pages (forms, and datagrids) from your database definitions.

O Adianti Framework está instalado!
Agora você já pode começar a criar suas aplicações, seguindo as etapas:

1) Crie o banco de dados do projeto;
2) Crie as classes de modelo para representar suas entidades;
3) Crie os controladores de página;
4) Customize o template da aplicação.

Adquira já o livro "Adianti framework para PHP" para aprender tudo sobre o framework, sua arquitetura, como criar as classes de modelo (Active Record), como criar as páginas, como usar seus componentes, como personalizar o template, como lidar com permissões, etc;

Utilize o Adianti Studio Professional para agilizar a criação da aplicação. O Studio Pro permite desenhar telas (formulários) para o Adianti Framework, bem como documentos PDF que podem ser usados para criação de relatórios. Também permite gerar automaticamente páginas (formulários e datagrids) a partir das definições do banco.
